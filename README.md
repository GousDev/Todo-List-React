
# Todo List in React
A simple Todo List react app for beginners

## Features
- Free to use

## Tech Stack
- React
- Vite

## How to use
1. Clone the repo
``` bash
git clone https://github.com/GousDev/Todo-List-React.git
```

2. Install dependencies
``` bash
npm install
```

3. Start the dev server
``` bash
npm run dev
```
